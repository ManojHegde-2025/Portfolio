## 📂 Project Structure

The project follows a modular React component structure, isolating core pages and layout architectures:

```text
portfolio/
├── public/                 # Static assets (favicons, SVGs)
├── src/
│   ├── assets/             # Images, logos, and vector graphics
│   ├── pages/
│   │   └── Home.jsx        # Main landing page compiling all sections
│   ├── App.jsx             # Root application component routing/calling Home.jsx
│   ├── index.css           # Global styles and Tailwind configuration
│   └── main.jsx            # Application entry point
├── package.json            # Dependencies and scripts
└── vite.config.js          # Vite build environment configuration
```

## ⚙️ Development Workflow
This repository uses a production-safe Git Feature Branch workflow combined with CI/CD automation:

* Local Development: Features are built and tested locally within VS Code.

* Staging: Changes are pushed to a development branch (dev).

* Preview: Vercel automatically generates an isolated Preview Deployment for testing the live site before going live.

* Production: Changes are manually reviewed via a GitHub Pull Request and merged into the main branch, triggering an instant production deployment.

## Local Setup Instructions
To run this project locally on your machine:

1. Clone the repository:
  Bash
  git clone [https://github.com/ManojHegde-2025/portfolio.git](https://github.com/ManojHegde-2025/portfolio.git)
  cd portfolio

2. Install dependencies:
  Bash
  npm install

4. Start the local development server:
  Bash
  npm run dev

4.Build for production:
  Bash
  npm run build
